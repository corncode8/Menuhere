# Menuhere
> 기존의 QR메뉴판을 응용한 QR주문 서비스

## 서비스 주요 기능
### 로그인/회원 가입
- Google, Github Oauth2 로그인 및 JWT 토큰을 사용하여 제공
- 회원가입 : 기본 회원가입 제공
## 메인 페이지
- 메뉴 조회
- 카테고리, 이름별 조회
- 메뉴 설명 제공
### 장바구니 페이지
- 세션으로 구현되어 비회원도 장바구니, 주문 가능
- 장바구니에서 수량 수정, 제거
### 주문 페이지
- 매장 내 식사, 배달 주문, 예약 제공
- KakaoPay, Payco 결제
### 주문 조회
- 상세 조회 기능 제공
